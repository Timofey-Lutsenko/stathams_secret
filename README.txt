Для установки необходимых зависимостей введите или скопируйте:

(venv) pip3 install -r requirements.txt

!!!Если вы запускаете проект в первый раз введите следующие команды!!!

(venv) python manage.py makemigrations
(venv) python manage.py migrate


Для запуска сервера введите или скопируйте:

(venv) python manage.py runserver

Для корректной работы программы на локальной машине должнен быть установлен сервер MongoDB

Для создания тестовой БД и ее заполнения запустите файл mainapp/common/db_filler.py

скрипт создаст тестовую коллекцию на локальном сервере MongoDB и заполнит данными из mainapp/common/samples.json

+++++++++++++++++++++++++++++++++++++++++++
+++               ОПИСАНИЕ              +++
+++++++++++++++++++++++++++++++++++++++++++

Программа получает пользовательский POST запрос по url и производит поиск по шаблонам, хранимым в базе данных.
После поиска программа вернет имя наиболее подходящего шаблона из базы.

Шаблон считается подходящим, если его поля полностью соотвествуют полям запроса, 
то есть имена полей запроса и тип хранимых в них данных, соотсетствуют полям и типам шаблона. 

В качестве входных параметров допускается передача большего количества полей, чем предполагают шаблоны,
в этом случае будет возвращено имя наиболее подходящего.

В случае отсутствия подходящего шаблона, программа вернет словарь, 
где ключами будут переданные поля, а значениями их тип данных.

Программа работает с 4-мя типами данных:
- date
- phone
- date
- str

В случае невозможности валидации значения передаваемого поля, ему присваивается тип str
